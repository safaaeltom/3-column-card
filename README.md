### What I learned

Through this project, I improved my understanding of Flexbox and responsive layout behavior, especially how to structure card components that adapt across different screen sizes.

A key learning point was realizing how typography and layout issues can sometimes be influenced by incorrect font loading, which affected spacing, wrapping, and overall visual consistency. Fixing the font import improved the accuracy of the design significantly.

I also gained more awareness of how to control text layout more effectively using width constraints instead of over-adjusting spacing manually.
